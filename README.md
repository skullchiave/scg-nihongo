# SCG日本語 ドリル

日本語学校の学習ドリル集。GitHub Pages で公開し、教室のプロジェクター投影と学生のスマホ両方で使う。

公開URL（学生に配る入口）: https://skullchiave.github.io/scg-nihongo/

## 構成

3カテゴリの母艦ポータル（`index.html`）から各ドリルへ進む：

- **教科書の練習** … 教材ひもづき（つなぐにほんご 等）→ `tsunagu/`
- **ドリル** … 教材によらない文法・語彙の横断練習（準備中）→ `general/`（予定）
- **問題** … 力だめし系（準備中）→ `quiz/`（予定）

各 HTML は 1 ドリルで自己完結（外部依存なし）。1ファイルでレスポンシブ両対応（PC/プロジェクター=投影用、スマホ=タップ用に自動切替）。

## ドリルの足し方（データ駆動）

`index.html` 冒頭の `DATA` に1行追加するだけで母艦にカードが出る。

```js
{tb:'tsunagu', lesson:'L7', label:'ない形', file:'tsunagu/L7-2.html', ready:true},
```

`ready:false` は「じゅんびちゅう」表示（押せない）。HTML を作って `ready:true` にすれば公開。

## 編集・公開

1. ローカルで HTML 編集（ブラウザでウィンドウを狭めるとスマホ用レイアウト確認可）
2. `git add` → `git commit` → `git push`（数十秒〜1分で URL 反映）

## フォルダ

- `index.html` … 母艦ポータル
- `tsunagu/*.html` … つなぐにほんごドリル本体
- `_original/`, `scratch/`, `tmp/` … 参照用・実験・一時（git 追跡しない）

## 公開方針（重要）

- 認証なし開放（中身は無害な練習素材）。**答案・名簿・クラス限定情報は置かない**（必要なら別途ログイン制の場所に作る）。
- QR表示ボタンは投影(PC幅)専用。スマホでは隠す。
- 拡散が問題化したら簡易合言葉を導入（半年ローテで卒業生・他校生を締め出す）。
